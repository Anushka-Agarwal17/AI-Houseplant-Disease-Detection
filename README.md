This project uses Deep Learning (CNN with MobileNetV2) to detect diseases in houseplants from leaf images and provide care recommendations.

## Features
- Leaf disease detection
- Confidence-based prediction
- Smart care recommendation system
- Flask web application

## Tech Stack
- Python
- TensorFlow / Keras
- Flask
- OpenCV

## How to Run
1. Install dependencies
2. Run `train_model.py` to train model
3. Run `app.py` to start web app
